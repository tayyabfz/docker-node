## Basic Commands

```bash
 docker run -it --name my-any_container_name real-container-name
```

```bash
 docker build -t any-name .
```
```bash
 docker compose up
```
```bash
 docker compose down
```
```bash
 docker compose up -d #background
```

```bash
 docker network inspect bridge
``` 
```bash
 docker network ls
```
